# 福井の地盤オープンデータ

## アプリ

- [福井のデータ500件だけ表示するサンプルアプリ](https://geofukui.github.io/opendata/)

## データ

- [ボーリング交換用データ XML](data/)
- [ボーリング交換用データ CSV](csv/)

* 現在、福井県周辺のみ取得

## ユースケース

- 検索したい（標高の最小、最大、砂・粘土、年別、スパイクデータの除去？）
- 地域を区切ってまとめてダウンロード
- XMLデータを柱状図ビューワーで見たい
- CSVデータがほしい
- 柱状図の絵が見えたらいい
- 土質試験一覧が見たい
